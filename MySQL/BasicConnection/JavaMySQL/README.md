Java com MySQL no Eclipse

Antes de começar o projeto execute o script 

dentro da pasta "sql" no MySQL

Softwares usados foram:

Wampserver

MySQL Workbench

********************************************************************************

acesse o site do mysql-connector-java :

	https://dev.mysql.com/downloads/connector/j/

Em "Select Operating System", selecione a opção "Platform Idependent"

Baixe a pasta com o seguinte titulo :

Platform Independent (Architecture Independent), ZIP Archive

Extraia os arquivos de dentro da pasta

********************************************************************************

Crie uma pasta chamada "lib" dentro da pasta do projeto

Arraste o "mysql-connector-java-5.1.45-bin.jar" dentro da pasta "lib"

********************************************************************************

Clique com o botão direito em cima do projeto

-> Selecione a Opção Properties

-> Selecione Java Build Path

-> Selecione Libraries

-> Clique em Add JARs...

-> Selecione o "mysql-connector-java-5.1.45-bin.jar" dentro da pasta "lib"

-> Clique em OK

-> Clique em Apply and Close
